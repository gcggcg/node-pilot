### 7.3 下一步操作

1. **部署到目标环境**
   ```bash
   # 复制项目到服务器
   scp -r node-pilot user@server:/opt/

   # 配置启动
   cd /opt/node-pilot/backend
   ./node-pilot --db ./data/servers.db --listen :8080
   ```

2. **添加真实服务器**
   - 访问 http://localhost:8080
   - 进入 "Servers" 页面，点击 "+ Add Server"
   - 填写真实的 SSH 服务器信息

3. **创建并执行脚本**
   - 进入 "Scripts" 页面，创建脚本
   - 进入 "Tasks" 页面，选择脚本和服务器，执行任务
   - 查看 "Output" 页面实时监控执行结果
